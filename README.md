# 🌞 SolarWay Advisor: Solar Site Analysis System

**SolarWay Advisor** is a data-driven decision support tool for solar energy investments. Inspired by the "HaetBit-Gil Plus" platform, it provides professional-grade feasibility reports.

## ✨ Key Features
- **BBC-Style UI**: Professional reporting interface for high readability.
- **Location Intelligence**: Analyzes solar potential and grid connectivity (KEPCO data integration).
- **Investment Verdicts**: Instant "Green/Yellow/Red" signals for quick site evaluation.
- **Pilot Scope**: Currently optimized for Sacheon-si, South Korea.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3 (Custom Variables, Flexbox/Grid)
- **Typography**: Noto Serif KR, IBM Plex Mono (for data-centric looks)


## 🚀 Quick Start
1. Clone this repository:
   `git clone https://github.com`
2. Open `index.html` in your browser.

## 🇰🇷 프로젝트 개요 (Korean Description)

**SolarWay Advisor**는 태양광 발전소 부지 매입 및 투자 의사결정을 돕는 **데이터 기반 입지 분석 솔루션**입니다. LS일렉트릭의 '햇빛길 플러스' 컨셉을 바탕으로, 복잡한 전력 계통 데이터와 수익성을 직관적인 리포트 형태로 제공합니다.

### ✨ 핵심 기능

*   **전문적인 뉴스 스타일 UI**: BBC 뉴스 리포트 형식을 차용하여 분석 결과의 신뢰도와 가독성을 극대화한 인터페이스를 제공합니다.
*   **전력 계통 연계 분석**: 사천시 파일럿 데이터를 바탕으로 한전(KEPCO) 선로 용량 및 변전소 접속 가능 여부를 시뮬레이션합니다.
*   **자동 투자 판정 시스템**: 입지 조건에 따라 '적격(Green)', '주의(Yellow)', '불가(Red)' 신호를 즉각적으로 생성하여 빠른 판단을 돕습니다.
*   **데이터 중심 시각화**: 'IBM Plex Mono' 폰트와 그리드 레이아웃을 활용하여 수치 데이터를 깔끔하고 정교하게 시각화합니다.

### 🛠️ 기술 스택

*   **언어**: HTML5, CSS3 (Custom Variables, Flexbox/Grid)
*   **사용 폰트**: 
    - *명조*: Noto Serif KR (전문적인 리포트 느낌)
    - *고딕*: Noto Sans KR (일반 텍스트 가독성)
    - *고정폭*: IBM Plex Mono (기술 수치 데이터)
